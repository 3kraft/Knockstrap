Popover

Examples

Usage

Usage with data-bind

Options

All of the options can be observables, also option's object can be observable too

Name Type Default Description
options object | observable Default values from bootstrap 3 Bootstrap options for popover. Please, see Bootstrap documentation
template string | observable undefined Name of template for popover content. If name is not specified, uses 'text' property from 'options' object
data object | observable undefined Data for template

Alert

Examples

Usage

This binding can be used with virtual elements.

All of the options can be observables.

Name Type Default Description
message string | observable empty string Text of alert message.
type string | observable 'info' Type of alert message. Possible values are 'info', 'warning', 'danger', 'success'. To specify your own type you should define css-styles for this type. For exmaple, for type 'my-custom-type', you shoud provide css class 'alert-my-custom type'
template string | observable 'alert' Name of template for alert content
data object | observable object with message and type properties Data for template

Name Type Default Description

body object | observable Template binding for modal body
footer object | observable Template binding for modal footer
